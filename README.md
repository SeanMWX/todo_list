# todo_list
OpenClaw skill repo for tracking local todo items, progress, completion, and archives in SQLite.

## Runtime Files

- `SKILL.md`
- `scripts/todo_list.py`
- `references/`

## Local Verification

- `python test/run_smoke.py`
- `python test/run_cli_tests.py`
- `python skill_creator/scripts/quick_validate.py .` when the local helper checkout is present

The helper repo under `skill_creator/` is local-only and ignored by `.gitignore`.
